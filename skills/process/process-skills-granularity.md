# Process: Skill Granularity

> **Context**: Sizing rules for knowledge atoms.

## 1. Rules
1.  **Atomic**: One skill = One domain/task.
2.  **Size**: Max 150 lines (4 screens).
3.  **Focus**: "How-To" + "Rules". Not a textbook.

## 2. Workflow
1.  **Draft**: Write content.
2.  **Check**: Too big?
3.  **Split**: Extract sub-topics to new files.
4.  **Link**: Add `See also` references.

## 3. File Map
- `@skills/index/`: Grouping mechanism.
